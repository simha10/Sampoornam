ACT AS:
A world-class Awwwards-level Creative Developer and Brand Experience Director, specializing in ultra-premium web design, Next.js, Framer Motion, GSAP, advanced scroll-based storytelling, and cinematic e-commerce interactions.

THE TASK:
Design and implement the component architecture and styles for a high-end, Apple-level scrollytelling landing page for "Sampoornam Foods", a luxury cloud kitchen specializing in South Indian cuisine, Hindu sweets, and Namkeens. 

TECH STACK INTENT:
Framework: Next.js 14 (App Router)
Styling: Tailwind CSS 
Animation: Framer Motion (for UI overlays) + GSAP ScrollTrigger (for sequence)
Rendering: HTML5 Canvas for image-sequence playback
State Management: Zustand (for cart)

VISUAL DIRECTION & BRAND AESTHETIC:
Overall Vibe: Cinematic, ultra-clean, Michelin-star culinary aesthetic mixed with Apple-level product reveal mechanics. 
Color Palette:
- Primary Dark Background: Deep charcoal/black (must perfectly match the edge of the video asset so the frame borders are 100% invisible).
- Sweets Theme (Dark Mode): #121212 background, white text, metallic gold #D4AF37 accents.
- Namkeens Theme (Warm Mode): Warm mustard #F3CA52 background, deep espresso text.
Typography: Playfair Display for dramatic, elegant headings. Inter or Geist for clean, readable body text and UI elements.

NAVBAR STRUCTURE & BEHAVIOR:
Ultra-minimal top navigation bar. Fixed/sticky at the top. Starts completely transparent. After a slight scroll, it fades into a glassmorphism (backdrop-blur) dark bar.
Content: Text logo "Sampoornam Foods" on the left. Links: "Menu", "Sweets", "Namkeens", "Catering" in the center. Cart icon with a dynamic item counter on the right.

CORE INTERACTION: SCROLL-LINKED IMAGE SEQUENCE
A central HTML5 Canvas sticky section with height ~400vh. Inside this space, a 210-frame image sequence plays as the user scrolls. The canvas width/height is 100% of the viewport, preserving aspect ratio.
SCROLL LOGIC AND STORYTELLING BEATS:
1. 0–20% scroll (The Foundation): Fully blank, wet green banana leaf on a dark floor. 
   - Copy (Centered, fading in): "Purity in every drop. Rooted in authentic traditions."
2. 20–45% scroll (The Spices): Red chilies, mustard seeds, and curry leaves slide into the frame. 
   - Copy (Left-aligned, glass card): "The perfect blend of spices. Sourced locally, hand-ground daily."
3. 45–75% scroll (The Feast): Rapid assembly of Vada, Dosa, Jalebi, Samosas, and steel tiffins flying in from all angles. 
   - UI: Keep overlays hidden to focus on the cinematic motion.
4. 75–100% scroll (The Resolution): All components lock into place, perfectly spelling out the "SAMPOORNAM FOODS" typography image. 
   - Copy (Centered at bottom): "A feast for the senses." 
   - CTA: Gold "Order Now" button.

THE DUAL-THEME PORTAL (POST-HERO):
Immediately below the scroll sequence, create a full-bleed 50/50 split-screen section.
Left Side: "Sweets" (Dark mode styling).
Right Side: "Namkeens" (Warm mode styling).
Add hover interactions: hovering one side slightly expands it, scales up the background image, and dims the opposite side.

THE CART & WHATSAPP CHECKOUT SYSTEM (NO PAYMENT GATEWAY):
Implement a Zustand store `useCartStore`. 
Create a slide-out Cart Drawer component.
Products must have quantity toggles allowing selection by weight (250g, 500g, 1kg) or by piece, dynamically updating the item price.
The Checkout Form inside the drawer requires: Name, Phone, Address, Delivery Time.
The "Place Order" button must NOT use Stripe/Razorpay. Instead, it must map over the cart items, calculate the total, format a clear, readable text message, and redirect the user's browser to the WhatsApp API: `wa.me/917007066735?text=[URL_ENCODED_ORDER_STRING]`.

Execute this with buttery-smooth transitions, zero-lag canvas rendering, hardware-accelerated scroll, and meticulous attention to negative space.
